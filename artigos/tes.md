
# O que é Software Open Source?

O **software open source** (código aberto) é um tipo de software cujo **código-fonte é disponibilizado publicamente**, permitindo que qualquer pessoa possa **estudar, modificar e redistribuir** o programa.

Este modelo promove colaboração, transparência e inovação tecnológica.

---

## Características principais

Os softwares open source normalmente possuem as seguintes características:

- Código-fonte acessível
- Licenças abertas (MIT, GPL, Apache, etc.)
- Desenvolvimento colaborativo
- Comunidades ativas
- Evolução contínua

---

## Exemplos de software open source

Alguns exemplos bastante conhecidos incluem:

- **Linux** – sistema operativo
- **Mozilla Firefox** – navegador web
- **LibreOffice** – suíte de produtividade
- **Git** – sistema de controlo de versões
- **MediaWiki** – software por trás da Wikipédia

---

## Vantagens do open source

### 1. Transparência
Qualquer pessoa pode verificar como o software funciona internamente.

### 2. Segurança
Falhas podem ser identificadas e corrigidas mais rapidamente pela comunidade.

### 3. Custo
Normalmente gratuito, reduzindo custos para indivíduos e empresas.

### 4. Flexibilidade
Pode ser adaptado para diferentes necessidades.

---

## Possíveis desvantagens

Apesar das vantagens, existem alguns desafios:

- Necessidade de conhecimento técnico
- Suporte pode depender da comunidade
- Nem sempre há documentação completa
